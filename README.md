### Instalação

1. Clonar o repositório
   ```sh
   git clone https://github.com/fmarquesfilho/projetoweb2.git
   ```
2. Compilar e rodar
   ```sh
   cargo run
   ```
3. Compilar e testar
   ```sh
   cargo test
   ```
